# 线段树

![Head Figure](https://pic2.zhimg.com/v2-5e9124a6147143e51cea46755e9a0398_1440w.jpg?source=172ae18b)

线段树（Segment Tree）几乎是算法竞赛最常用的数据结构了，它主要用于维护区间信息（要求满足结合律）。与树状数组相比，它可以实现 $O(n log n)$ 的区间修改，还可以同时支持多种操作（加、乘），更具通用性。
