# 堆

一个堆(heap) 是一种特殊的基于树的数据结构，它满足下面描述的堆属性。

在一个最小堆 (min heap) 中, 如果 P 是 C 的一个父级节点, 那么 P 的 key (或 value )应小于或等于 C 的对应值。

在一个最大堆 (max heap) 中, P 的 key (或 value )大于 C 的对应值。
